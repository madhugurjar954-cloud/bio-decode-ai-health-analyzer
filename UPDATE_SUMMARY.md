## 🎓 HOW TO USE THE NEW VERSION

### Setup (5 minutes)
```bash
git clone https://github.com/madhugurjar954-cloud/bio-decode-ai-health-analyzer.git
cd bio-decode-ai-health-analyzer
```

- Change medical ranges
