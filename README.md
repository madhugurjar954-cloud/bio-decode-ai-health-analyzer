# Bio Decode AI — Health Analyzer

🧬 **Understand Your Lab Reports Instantly**

An AI-powered lab report analyzer that provides personalized health scores, medical insights, and nutrition plans in multiple languages.

## 🚀 Features

- **AI-Powered Analysis** - Uses Groq's Llama 3.3 70B model
- **Multi-Language Support** - English, Hindi, Punjabi, Gujarati, Spanish, French, Arabic
- **Health Score** - Personalized health assessment (0-100)
- **Detailed Explanations** - Medical info for each test
- **Nutrition Plans** - Personalized dietary guidance
- **Image Recognition** - Upload lab report photos for automatic extraction
- **Doctor Consultation Tips** - When to see a healthcare provider
- **100% Private** - No data storage, all processing is local

## 🌐 Live Demo

🔗 **[Open Bio Decode AI](https://bio-decode-ai-health-analyzer.streamlit.app)**

## 📋 Supported Lab Tests

- Thyroid (TSH, T3, T4)
- Diabetes (Blood Sugar, HbA1c)
- Anemia (Hemoglobin, Hematocrit, Ferritin)
- Kidney Function (Creatinine, BUN, eGFR)
- Liver Function (SGPT, SGOT, Bilirubin)
- Full Panel Analysis

## 🛠️ Technology Stack

- **Frontend:** Streamlit
- **AI Model:** Groq (Llama 3.3 70B)
- **Language:** Python
- **Deployment:** Streamlit Cloud

## 📦 Installation

```bash
# Clone repository
git clone https://github.com/madhugurjar954-cloud/bio-decode-ai-health-analyzer.git
cd bio-decode-ai-health-analyzer

# Install dependencies
pip install -r requirements.txt

# Set environment variable
export GROQ_API_KEY="your-api-key-here"

# Run locally
streamlit run app.py
```
